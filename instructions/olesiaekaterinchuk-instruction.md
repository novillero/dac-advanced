# Авторизация в Telegram
Это пошаговая инструкция о функциональности Telegram
Эта инструкция создана в рамках мастер-класса по docs as code.gi

1. Откройте приложение Telegram или https://web.telegram.org  
2. Введите номер телефона в международном формате (`+7XXXXXXXXXX`)  
3. Нажмите **Далее**  
4. Введите код подтверждения, полученный в Telegram или по SMS  
5. После подтверждения вы будете авторизованы

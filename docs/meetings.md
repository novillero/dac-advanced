#### Вебинар №2 «GIT для технического писателя»

Теоретическая часть:

1. Локальные, централизованные и распределённые системы контроля версий.
2. Что такое GIT?
3. Как работает GIT? Коммиты, ветки и операции слияния.
4. Уровни хранения данных в GIT.
5. Веб-платформы для управления проектами в GIT.

Практическая часть:

7. Git-медиум для технического писателя: некоторые нестандартные операции в GIT (отмена изменений, работа с коммитами и ветками, решение конфликтов слияния).

# Домашнее задание №6 «Настройка стилей док.проекта и использование единого источника»

### Стили док.портала

1. Подключите к проекту файл `extra.css` по [инструкции](../pet-project/mkdocs-css.md#extra).
2. Создайте свою цветовую схему и подключите её вместо светлой световой схемы в конфиге `mkdocs.yml` по [инструкции](../pet-project/mkdocs-css.md#scheme).
3. Настройте стили основных шрифтов по [инструкции](../pet-project/mkdocs-css.md#fonts).

### Единый источник

1. Подключите расширение `pymdownx.snippets`.
2. Создайте ресурсный файл, разместите в нём фрагмент для переиспользования (например, уведомление) и добавьте его в избранные статьи:

    ```title="Пример"
    !!! note "Примечание"
        Это уведомление добавлен с помощью расширения `pymdownx.snippets`
    ```

Для выполнения задания воспользуйтесь [инструкцией](../pet-project/snippets.md).

### Проверка домашнего задания

Пришлите в чат курса ссылку на статью с переиспользованным контентом.

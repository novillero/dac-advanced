# Домашнее задание №1 «Языки разметки»

1. Познакомьтесь с [базовым синтаксисом Markdown](../markdown.md).
2. Напишите пошаговую инструкцию для любого пользовательского сценария в телеграм и оформите её с помощью разметки markdown. 

    Пользовательский сценарий можно выбрать любой: как отправить отложенное сообщение, как создать папку с чатами, как изменить аватарку, как авторизоваться в десктоп-приложении, etc. Постарайтесь, чтобы было не меньше 4-5 пунктов.

!!! note "Примечание"
    Инструкция должна быть оформлена в виде markdown-файла (расширение `.md`) С такими файлами удобно работать в **VS Code**.

    Также для самопроверки вы можете воспользоваться встроенной в VS Code функцией предпросмотра md-файлов. Откройте файл в VS Code и нажмите комбинацию клавиш `Ctrl+Shift+V` или `Cmd+Shift+V`.

# Домашнее задание №0 «Настройка окружения»

Для практической работы вам понадобятся установленные на компьютер **python** и **git**, аккаунт на **GitHub** и редактор кода **Visual Studio Code**.

1. Python — это язык программирования, на котором написан генератор статических сайтов MkDocs, с которым мы будем работать. Перейдите на [оф.сайт](https://www.python.org/downloads/){target=_blank}, выберите стабильную версию (3.10 и выше), скачайте дистрибутив и установите себе на устройство.
2. VS Code — это редактор кода, предназначенный для работы на разных языках программирования. Он отлично подходит для работы с языками разметки, поэтому я рекомендую использовать его. Перейдите на [оф.сайт](https://code.visualstudio.com/Download){target=_blank}, выберите вашу ОС, скачайте нужный дистрибутив и установите редактор себе на компьютер.
3. GIT — это система контроля версий, одна из ключевых программ для работы в парадигме docs as code. Перейдите на [оф.сайт](https://git-scm.com/install/){target=_blank}, выберите нужную операционную систему, скачайте Standalone installer (для Windows) или установите GIT по инструкции (для macOS и Linux). В процессе установки в качестве используемого редактора выберите VS Code.
4. GitHub — это web-платформа для работы с git-репозиториями. Если у вас ещё нет аккаунта: перейдите на [https://github.com/](https://github.com/){target=_blank}, нажмите кнопку **sign up** и зарегистрируйтесь.

### Проверка домашнего задания

1. Откройте командную строку (CMD на Windows, Bash на Linux) и последовательно выполните две команды:

    ```
    git --version
    ```

    ```
    python --version
    ```

    Если git и python установились корректно, в ответе на команды вам отобразятся версии установленного по:

    ```
    C:\Users\User1>python --version
    Python 3.10.2

    C:\Users\User1>git --version
    git version 2.33.0.windows.2
    ```

2. Пришлите в телеграм-чат **Docs as code PW-1** скопированный из командной строки результат выполнения команд (можно скриншотом, можно `ctrl-v` + `ctrl-c`) и ссылку на ваш github-профиль.

Если что-то не получается или возникают вопросы, пишите в чат.

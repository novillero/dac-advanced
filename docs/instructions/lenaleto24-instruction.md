# Отправить отложенное сообщение в Telegram

Эта инструкция создана в рамках мастер-класса по docs as code.

## Пошаговая инструкция

1. Откройте Telegram.
   
2. Перейдите в нужный чат — личный диалог, группу или канал.
   
3. Введите текст сообщения.
   
4. Откройте меню отложенной отправки:  
   
   - на смартфоне: нажмите и удерживайте кнопку отправки (иконка самолетика).
  
   - на компьютере: нажмите правой кнопкой мыши по кнопке отправки.
  
5. Выберите пункт **Отправить позже**. 
   
6. Установите дату и время, когда сообщение должно быть отправлено.
   
7. Подтвердите выбор.

## Результат

Сообщение будет автоматически отправлено в выбранное время без дополнительных действий.

Для проверки отложенного сообщения нажмите на иконку календаря в поле ввода сообщения.

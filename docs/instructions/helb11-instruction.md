# Создание папок в мобильном приложении Telelgram

!!! note "Примечание"
	Эта инструкция создана в рамках мастер-класса по docs as code.

!!! note "Примечание"
	Это пошаговая инструкция о функциональности Telegram.

1. Зайдите в меню нажав на кнопку в правом верхнем углу экрана. 

2. Перейдите в раздел **Настройки**.

3. В открывшемся окне выбирите пункт **Папки с чатами**.

	Вы попали в основное меню управления папками.

4. Для создания папки достаточно выбрать пункт **Создать новую папку**.

5. В открывшемся окне настройки папки Вы можете: 

    1. Задать название папки. 

	2. Выбрать чаты для включения в папку. 

	3. Чаты, которые не должны отображаться в папке.

	4. Цвет папки в списке чатов. 

   Также вы можете создать ссылку-приглашение и поделиться своей папкой с другими пользователями.

!!! warning "Внимание"
	Для создания папки как минимум нужно ввести её  и добавить хотя бы один чат.
	
6. Нажимите кнопку **Сохранить** в правом верхнем углу и папка готова.

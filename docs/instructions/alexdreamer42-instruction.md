# Отправить отложенное сообщение

Это пошаговая инструкция о функциональности Telegram
Чтобы отправить отложенное сообщение:
1. Выберите получателя.
2. Введите текст сообщения, но не отправляйте его.
3. Нажмите правой кнопкой мыши на значок отправки сообщения.
4. Выберите пункт **Schedule message**.
5. Выберите дату и время отправки. 
6. Если у вас есть Telegram Premium, вы можете отправить сообщение многократно с определённой периодичностью. Для этого выберите периодичность повтора в поле **Repeat**.
7. Нажмите кнопку **Schedule**.

Сообщение появится в разделе **Scheduled messages**. Пока сообщение не отправилось, вы можете его отредактировать, удалить, переназначить на другое время или отправить мгновенно.

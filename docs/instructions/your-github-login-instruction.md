# Отправка вложенных материалов в Telegram

## Общие сведения

При отправке личного сообщения в Telegram на смартфоне можно прикрепить следующие форматы данных: 

- изображение
  
- видео
  
- файл
  
- геопозиция
  
- контакт
  
- аудио-файл

## Отправка изображения/видео 

Для отправки изображения/видео в личном сообщении в Telegram выполните следующие шаги:

1. Нажмите на значок скрепки.
   
2. В нижней панели выберите вкладку «Галерея».

Если необходимо сделать изображение/видео сразу, нажмите на значок камеры. 

Для выбора изображения/видео из галереи смартфона:

1. Откройте вкладку «Галерея».
   
2. Выберите желаемое изображение или видео (удерживайте для множественного выбора).
   
3. Подтвердите выбор файла в чате.

### Отправка файла

Для отправки файла в личном сообщении в Telegram необходимо выполнить следующие шаги:

1. Нажмите значок скрепки.
   
2. В нижней панели выберите вкладку «Файл».

После выбора вкладки «Файл» отобразится панель визуально разделенная на две части: 

- Верхняя часть содержит список папок-директорий, из которых можно выбрать файл для отправки.
  
- Нижняя часть содержит список файлов, которые были недавно отправлены через Telegram, для быстрого повторного использования.

Выберите файл и подтвердите выбор в чате. 

### Отправка геопозиции

Для отправки геопозиции в личном сообщении в Telegram  выполните следующие шаги:

1. Нажмите на значок скрепки.
   
2. В нижней панели выберите вкладку «Геопозиция».

После выбора вкладки «Геопозиция» отобразится панель визуально разделенная на две части: 

1. Верхняя часть содержит интерактивную карту, на которой можно выбрать геопозицию, а также две кнопки:

    - **«Отправить выбранную геопозицию»** - для отправки выбранного местоположения.

    - **«Транслировать геопозицию»** - для обмена своей текущей геопозицией в режиме реального времени.
  
1. Нижняя часть содержит список мест поблизости, что облегчает поиск и отправку известных мест.

После нажатия кнопки **«Транслировать геопозицию»** отобразится окно, в котором выберите, как долго участники чата смогут видеть геопозицию. Доступные варианты:

- 15 минут
  
- 1 час
  
- 8 часов
  
- трансляция будет активна, пока пользователь не отключит ее вручную

### Отправка контакта

Для отправки контакта в личном сообщении в Telegram выполните следующие шаги:

1. Нажмите на значок скрепки.
   
2. В нижней панели выберите вкладку «Контакт».
   
3. Введите в поле поиска наименование контакта или выберите вручную из списка.
   
4. Подтвердите выбор в чате.

### Отправка аудио-файла

Для отправки аудио-файла в личном сообщении в Telegram выполните следующие шаги:

1. Нажмите на значок скрепки.
   
2. В нижней панели выберите вкладку «Музыка».
   
3. Введите в поле поиска наименование аудио-файла, сохраненного в памяти смартфона, или выберите вручную из списка.
   
4. Подтвердите выбор в чате.

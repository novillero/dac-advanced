# Отправка вложенных материалов в Telegram-чате

## Общие сведения

При отправке личного сообщения в Telegram-чате на смартфоне можно прикрепить следующие форматы данных: 
- изображение
- видео
- файл
- геопозиция
- контакт
- аудио-файл

## Отправка изображения/видео 

Для отправки изображения/видео в личном сообщении в Telegram-чате необходимо выполнить следующие шаги:
1. Нажать значок скрепки
2. В нижней панели выбрать вкладку "Галерея"

Если необходимо сделать изображение/видео сразу, нажмите на значок камеры. 

Для выбора изображения/видео из галереи смартфона, необходимо:
1. Открыть вкладку "Галерея"
2. Выбрать желаемое изображение или видео (удерживайте для множественного выбора)
3. Подтвердить выбор файла в чате

### Отправка файла

Для отправки файла в личном сообщении в Telegram-чате необходимо выполнить следующие шаги:
1. Нажать значок скрепки
2. В нижней панели выбрать вкладку "Файл"

После выбора вкладки "Файл" отобразится панель визуально разделенная на две части: 
- верхняя часть содержит список папок-директорий, из которых можно выбрать файл для отправки
- нижняя часть содержит список файлов, которые были недавно отправлены через Telegram-чат, для быстрого повторного использования

Выберите файл и подтвердите выбор в чате. 

### Отправка геопозиции

Для отправки геопозиции в личном сообщении в Telegram-чате необходимо выполнить следующие шаги:
1. Нажать значок скрепки
2. В нижней панели выбрать вкладку "Геопозиция"

После выбора вкладки "Геопозиция" отобразится панель визуально разделенная на две части: 
1. Верхняя часть содержит интерактивную карту, на которой можно выбрать геопозицию, а также две кнопки: 
- "Отправить выбранную геопозицию" - для отправки выбранного местоположения
- "Транслировать геопозицию" - для обмена своей текущей геопозицией в режиме реального времени
2. Нижняя часть содержит список мест поблизости, что облегчает поиск и отправку известных мест

После нажатия кнопки "Транслировать геопозицию" отобразится окно, в котором необходимо выбрать, как долго участники чата смогут видеть геопозицию. Доступные варианты:
- 15 минут
- 1 час
- 8 часов
- трансляция будет активна, пока пользователь не отключит ее вручную

### Отправка контакта

Для отправки контакта в личном сообщении в Telegram-чате необходимо выполнить следующие шаги:
1. Нажать значок скрепки
2. В нижней панели выбрать вкладку "Контакт"
3. Ввести в поле поиска наименование контакта или выбрать вручную из списка
4. Подтвердить выбор в чате

### Отправка аудио-файла

Для отправки аудио-файла в личном сообщении в Telegram-чате необходимо выполнить следующие шаги:
1. Нажать значок скрепки
2. В нижней панели выбрать вкладку "Музыка"
3. Ввести в поле поиска наименование аудио-файла, сохраненного в памяти смартфона, или выбрать вручную из списка
4. Подтвердить выбор в чате

# **Отправка отложенного сообщения в Telegram Desktop**

Эта инструкция создана в рамках мастер-класса по docs as code.

1. Напишите сообщение.

2. Нажмите правой кнопкой мыши на кнопку отправки сообщения.

3. В выпадающем меню выберите «Отправить позже» («Schedule Message»).

4. Выберите дату (кликните по дате левой кнопкой мыши и выберите в календаре нужную дату) и время отправки (задайте время на строке со временем).

5. Нажмите на кнопку **Запланировать**.

В назначенную дату получателю придет сообщение.

## *Отмена отправки отложенного сообщения*

1. Напишите сообщение.

2. Нажмите правой кнопкой мыши на кнопку отправки сообщения.

3. В выпадающем меню выберите «Отправить позже» («Schedule Message»).

4. Нажмите на кнопку **Отмена**.

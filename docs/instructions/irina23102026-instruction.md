## 1. Приложение Telegram «Секретные чаты»

Приложение Telegram «Секретные чаты» позволяет вести диалог, скрывая содержимое переписки от третьих лиц.  

Все текстовые сообщения и файлы (изображения, видео и др.) передаются с использованием **сквозного (end-to-end) шифрования**: ключ шифрования доступен только на устройствах собеседников.

### Особенности секретных чатов:

* Содержимое не хранится на серверах Telegram. 

* Переписка доступна только на одном устройстве.

* При выходе участника из аккаунта Telegram чат удаляется.

* Невозможно переслать сообщения или сделать скриншот. 

* Можно настроить автоматическое удаление сообщений и файлов.  

* Удалённые диалоги невозможно восстановить.  

### Дополнительные функции:

* Включение/отключение уведомлений.  

* Использование видеозвонков.  

* Очистка истории.  

* Удаление чата.

## 2. Приложение Telegram «Секретные чаты» на устройстве Android

> **Важно:**  

> - При выходе из аккаунта Telegram секретный чат будет удалён.
>
> - Секретный чат, созданный на одном устройстве, недоступен на других.

### 2.1. Создание секретного чата

1. Откройте Telegram.  

2. Нажмите кнопку **Добавить**.  

3. В меню выберите **Создать секретный чат**. 

4. Выберите контакт для переписки.  

> **Примечание:**  

> - Если контакт находится в статусе **Не в сети**, создание чата будет отложено до тех пор, пока он не станет **В сети**. Появится соответствующее уведомление.

Если контакт **В сети**, откроется окно нового секретного чата.

**Внешний вид секретного чата в списке:**

- Зелёная подсветка.

- Иконка замка рядом с названием.

> **Примечание:**

> - С одним и тем же контактом можно создать несколько секретных чатов.

---

### 2.2. Контроль переписки

В секретных чатах действуют ограничения конфиденциальности:

- Кнопка **Переслать** заблокирована.  

- Запрещены **скриншоты** и **запись экрана**.  

  - При попытке сделать скриншот появится предупреждение. 

  - При записи экрана содержимое чата скрывается.

---

### 2.3. Настройка таймера для удаления сообщений/файлов

Можно настроить автоматическое удаление сообщений и файлов через интервал от **1 секунды до 1 недели**.

#### 2.3.1. Включение таймера для удаления сообщений/файлов

1. Перейдите в меню.  

2. Выберите **Включить удаление по таймеру**. 

3. Укажите желаемое время и нажмите **Готово**.

**После настройки:**

- Возле аватара появится значок с временем.  

- В чате появится системное уведомление о включении таймера.

> **Важно:**  

> - Таймер начинает отсчёт **после прочтения сообщения** или получения файла собеседником.  
>
> - Удаляются **только сообщения и файлы, отправленные после включения таймера**.

#### 2.3.2. Выключение таймера для удаления сообщений/файлов

1. Перейдите в меню.  

2. Снова выберите **Включить удаление по таймеру** (это переключатель).  

3. Таймер будет отключён.

**После отключения:**

- Значок времени исчезает.  

- В чате появляется уведомление об отключении таймера.

---

### 2.4. Удаление секретного чата

1. Перейдите в секретный чат. 

2. Перейдите в меню.  

3. Выберите **Удалить чат**.  

Появится запрос на подтверждение:

- **Удалить только у себя** — оставьте чекбокс **Также удалить для…»** без изменений
и нажмите кнопку **Удалить чат**.  

- **Удалить у обоих собеседников** — установите галочку в чекбоксе **Также удалить для…»** и нажмите кнопку **Удалить чат**.

После подтверждения чат будет удалён с устройства.
